# Clone da Landing Page da Uber

Projeto desenvolvido durante a **sexta semana da capacitação** da área de **Projetos Desktop da Asimov Jr.**

O objetivo desta atividade foi aprofundar os conhecimentos em **Tailwind CSS** por meio da recriação da landing page da Uber. Durante o projeto, foram aplicados conceitos de responsividade, organização de layouts, componentes visuais e estilização utilizando classes utilitárias, resultando em uma interface moderna e inspirada no design da plataforma.

---

## Tecnologias utilizadas

- HTML5
- Tailwind CSS

---

## Funcionalidades

- Header responsivo com menu de navegação;
- Banner principal com chamada para ação;
- Cards informativos sobre os serviços da plataforma;
- Seções institucionais e de segurança;
- Área para download dos aplicativos via QR Code;
- Footer com links para as lojas de aplicativos;
- Layout responsivo para dispositivos móveis e desktop;
- Efeitos de hover e transições utilizando Tailwind CSS.

---

## Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados conceitos como:

- Estruturação de páginas com HTML5;
- Estilização utilizando classes utilitárias do Tailwind CSS;
- Organização de layouts com Flexbox e Grid;
- Responsividade com breakpoints (`sm`, `md` e `lg`);
- Utilização de ícones com Font Awesome;
- Aplicação de transições, animações e efeitos de hover;
- Construção de interfaces inspiradas em aplicações reais.
